# To-Do List Java

Projeto simples de uma To-Do List em Java (console) criado para estudar POO, coleções e persistência em arquivo.

## Tecnologias
- Java 21
- IntelliJ IDEA

## Como rodar
1. Abra o projeto no IntelliJ.
2. Execute a classe `br.com.marco.todolist.Main`.

## Funcionalidades
- Adicionar, listar, concluir e remover tarefas
- Persistência em arquivo `tarefas.txt`
